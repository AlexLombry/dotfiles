# Upd8R
Script Based on Keep Your macOs Stuff Updated (KYMSU)

> Get your Homebrew, Mac App Store, Atom Package Manager (apm), npm, yarn, composer, rustup ... stuff updated.

## Requirements

- [Homebrew](https://brew.sh)

## Usage

Only update all the things
`$ upd8r`

With cleanup after updates
`$ upd8r cleanup`

## Plugins system

All default plugins are placed in `~/.upd8r/plugins.d`, feel free to delete those you don't use, and/or add new ones.
Don't forget to share them here with a shiny pull request! âœ¨

## Auto-update

Upd8R will auto-update itself each time you run `upd8r` command.
